# Datasets for TruthLens

This folder contains datasets used for fake news detection.

## 1. LIAR Dataset
- Source: [LIAR Dataset on Kaggle](https://www.kaggle.com/datasets/mrisdal/fake-news)
- Format: CSV
- Description: Short statements labeled for truthfulness.
- Usage: Preprocessing done in backend/utils/preprocessing.py

## 2. Kaggle Fake News Dataset
- Source: [Kaggle Fake News Dataset](https://www.kaggle.com/c/fake-news/data)
- Format: CSV
- Description: News articles labeled as fake or real.

## 3. FakeNewsNet
- Source: [FakeNewsNet](https://github.com/KaiDMML/FakeNewsNet)
- Description: Includes news content and social context for each article.
