# Frontend Prototype for TruthLens

This folder contains the **frontend prototype** of the TruthLens system.

## Contents

- React / HTML / CSS / JS code for the six screens
- `screenshots/` : Folder to store screenshots of the prototype

## Six Screens

1. **Home / Landing Page** – Entry point of the system
2. **Upload News / Article** – Upload news content for prediction
3. **LIME Prediction** – Shows model prediction with LIME explainability
4. **SHAP Prediction** – Shows model prediction with SHAP explainability
5. **Evaluation Dashboard** – Visualizes metrics, past predictions, and system evaluation
6. **About / Info Screen** – Details about TruthLens system and team

## Usage

- This frontend is implemented in **React / HTML / CSS / JS**  
- Screenshots of each screen are saved in the `screenshots/` folder  
- Navigate between screens to demo the system
